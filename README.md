# RD Digital Platform 管理后台原型

本仓库包含基于 Bootstrap 构建的 **RD Digital Platform** 管理后台高保真可交互原型。

## 功能概览

- 标准后台框架：顶部栏 + 侧边导航 + 多标签内容区。
- 首页仪表盘：展示租户、应用、云资源关键 KPI 与活动。
- 租户管理：列表、成本详情、添加/编辑租户模态框。
- 应用管理：按租户查看应用与云资源，创建/编辑应用（含 Terraform 编辑模态）。
- 云资源总览：跨租户资源运行状态与成本表格。
- 模板管理：模板列表、版本更新、创建模态框。
- 角色管理：角色列表、添加/编辑、授权用户模态。
- 知识库：文档列表与上传模态。
- QnA 聊天：点击图标弹出聊天面板，可进行问答互动。

## 快速预览

直接使用浏览器打开 `index.html` 即可体验原型：

```cmd
start index.html
```

> 上述命令适用于 Windows CMD。也可以双击 `index.html`。

## 结构

```text
index.html              # 主页面与所有模态、内容模板
assets/
  css/style.css         # 自定义样式，覆盖 Bootstrap 默认风格
  js/main.js            # 页面逻辑：导航、标签页、聊天、资源联动
```

## 后续可扩展方向

- 将模拟数据替换为真实 API 对接。
- 集成图表库（ECharts、Chart.js 等）渲染真实趋势与成本图表。
- 引入前端框架（例如 React、Vue）重构以支持状态管理与路由。
- 将 QnA 聊天对接实际的知识库检索或 LLM 服务。
